# Unified Metadata Schemas

PreGod enables developers to swiftly plug in the data returned by its APIs, without worrying about the underlying structure. We take care of all the dirty work to provide Unified Metadata Schemas (UMS) for all `type`.

Below you will see a list of types and the corresponding UMS.

:::info
This page is still being revised, the UMS listed above may subject to future changes.
:::

## Transaction

All types under the tag `transaction` (`transfer`, `mint`, `burn`, `self`) share the same UMS:

```json
{
    "logo": "https://s2.coinmarketcap.com/static/img/coins/64x64/3890.png",
    "name": "Polygon",
    "symbol": "MATIC",
    "decimals": 18,
    "token_value": "1000000000000000000",
    "token_address": "0x0000000000000000000000000000000000001010",
    "token_standard": "ERC-20"
}
```

## Social

<Tabs>
  <TabItem value="social_post" label="post/revise">

`post` indicates an action of publishing a new post.

`revise` indicates an action of updating an existing post.

```json
{
    "title": "Guess what's coming",
    "body": "Guess what's coming",
    "media": [
        {
            "address": "https://ipfs.infura.io/ipfs/QmXGp2tq7yTXMoeKbDqAsFNW9Qw1GGzWbEFiXmhKxVikGj",
            "mime_type": "image/png"
        }
    ],
    "type_on_platform": "Post"
}
```

</TabItem>

<TabItem value='social_comment_share' label='comment/share'>

`comment` indicates a comment on an existing post.

`share` indicates a sharing of an existing post, think of it as a retweet.

```json
{
    "body": "RSS3 is building a Lens integration!",
    "target": {
        "body": "Yo! from RSS3! https://rss3.io",
        "type_on_platform": "Post"
    },
    "type_on_platform": "Comment"
}
```

</TabItem>

  <TabItem value="social_profile" label="profile">

`profile` indicates an action of creating a new credential.

```json
{
    "bio": "The awesome Henry.",
    "name": "Henry",
    "handle": "Henry",
    "source": "crossbell",
    "network": "crossbell",
    "platform": "",
    "profile_uri": ["ipfs://QmSX9QiwjTGBk5m22UscTg3vrbMwUfFsmxVzMH57hkPD5U/1021.png"]
}
```

</TabItem>
